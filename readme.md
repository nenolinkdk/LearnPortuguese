# Learn Portuguese

Little Android app for Danish beginners learning Portuguese.

The app presents one easy Portuguese phrase at a time with:

- Danish translation
- beginner grammar note
- small glossary
- previous/next phrase navigation
- Android TextToSpeech button to pronounce the Portuguese phrase
- visible on-screen disclosure: "AI used: beginner phrase set and grammar notes"

The first phrase deck uses common beginner vocabulary and stays close to a basic 1000-word learning level.

## Open

Open this folder in Android Studio, let Gradle sync, then run the `app` configuration on an Android device or emulator. Portuguese pronunciation depends on the Android device having a Portuguese TextToSpeech voice installed.
